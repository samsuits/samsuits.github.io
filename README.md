# samsuits.github.io

This repository contains the source for the personal website `samsuits.github.io`.

Remote: https://github.com/samsuits/samsuits.github.io.git

Quick commands

```bash
# stage and commit
git add .
git commit -m "Update site"

# push current branch to the configured origin
git push
```

The `main` branch is set to track `origin/main`.
